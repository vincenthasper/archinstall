# archinstall (unmaintained)

Just a simple Arch Linux installation script based on my personal preferences. However, I no longer actively use this script anymore, as I now use the default Arch Linux installation script provided with the Arch ISO. Nevertheless, the script still works, and I have kept it here for learning purposes.

### ⚠️ Important

If you are considering using this script, be aware that it will fully wipe out your disk without any confirmation prompt and does not ask for confirmation when entering details such as disk selection, hostname, root password, username, or user password.

### 🚀 Usage

Make sure you are connected to the internet within the Arch ISO environment, then run:

```bash
bash <(curl -L https://raw.githubusercontent.com/vincenthasper/archinstall/main/archinstall.sh)
```
